**Примечания:**
- Все уязвимости эксплуатируются через **страничный кэш** (page cache) и приводят к подмене содержимого файлов при последующих чтениях.
- Основной вектор — модификация SUID-root бинарников или критических файлов (`/etc/passwd`).
- Рекомендуется обновить ядро до последних стабильных версий и применить митигации по отключению ненужных модулей.
